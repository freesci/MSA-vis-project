Installation termcolor
----------------------

1.download termcolor package from http://pypi.python.org/pypi/termcolor bottom of the page
2.go to the folder where the package is
3.type python in terminal 
4.type:
tar xzvf termcolor-*.tar.gz.
cd termcolor-*.
sudo python setup.py install.

Installation argparse
---------------------

You may download argparse from http://argparse.googlecode.com/files/argparse-1.2.1.tar.gz
Make all the same steps like during install termcolor.
